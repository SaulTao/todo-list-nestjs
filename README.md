# TODO List NestJS

## Description

用Nest.js 设计一套todo list的接口
1. 使用 nest.js
2. todo可以添加修改删除，标记是否完成
3. 接口符合restful标准
4. 完成接口controller层测试


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
